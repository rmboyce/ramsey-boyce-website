A speed tracker for the Rubik's Cube

Overview

Built with Python and PyQt, this Rubik's Cube timer tracks solve times and logs them to a text file with timestamps for progress tracking.

Link to get it: https://github.com/rmboyce/cube-timer/releases/

Link to see the code: https://github.com/rmboyce/cube-timer

Usage

Press and release the spacebar to start timing (runs on release), and press it again to stop. Afterwards, press space to log the time, clear it, or mark it as DNF.

The stats page shows session statistics, and the PLL tab lists algorithms for permuting the last layer of the Rubik's Cube.
