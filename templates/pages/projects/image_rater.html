An image rater based on the average person's judgment

Overview

This Python program trains a neural network to score image aesthetics based on human preferences, using the AVA (Aesthetic Visual Analysis) dataset (available here).

View the Jupyter Notebook PDF here.

Usage

Download the Jupyter Notebook (`.ipynb`) file from GitHub or view the PDF version here. Running it locally requires downloading the AVA dataset to the path specified in the code (default is `E:/AVA_dataset`).

Technical Details

The model uses a pre-trained VGG19 network to convert images into feature codes, which are then passed to a two-layer fully-connected neural network. The output rating is a probability distribution from 1 (lowest) to 10 (highest quality).

It achieves a 0.52 correlation coefficient on validation data. Factor analysis identifies VGG19 codes with the largest impact on aesthetics, and helper functions visualize images high in specific activations.
