A place for my projects using Flask, HTML, and Javascript

Overview

This website is a place for me to put my projects when I complete them. I thought that making a website would be fun, and as a bonus I could share the stuff I've made.

Technical Details

I use Flask as a backend framework along with HTML and CSS to make the web pages. I don't use any frontend framework, because I haven't found one that I like yet. My p5.js visualizations are made with Javascript.

My website is hosted on Google Cloud's App Engine.
