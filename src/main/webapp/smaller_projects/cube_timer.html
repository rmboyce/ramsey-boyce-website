Rubik's Cube Timer

  • A speed tracker for the Rubik’s Cube

Overview

  • This is a Rubik’s Cube timer made using Python and PyQt to track how fast the user can solve a cube. The program also saves every past logged time in a text file with timestamps, so one can see their improvement in the long run.
  • Link to check it out and get the code: https://github.com/rmboyce/cube-timer

Usage

  • To operate it, press and release the spacebar to start the timer (time starts on release) and press the spacebar again to stop the timer. Then one can decide whether to log the time (by pressing space), clear the time, or mark it as a DNF (did not finish: stopped the timer before the cube was completely solved).
  • The stats page shows statistics on the previous times in the session (from when the program is opened). The PLL tab just shows some algorithms to permute the last layer of the Rubik’s cube. I thought that it might be nice if one is new to the fast method of solving the cube and needs to jog their memory on the PLL algorithms.
