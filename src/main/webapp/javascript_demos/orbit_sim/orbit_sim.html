Orbit Simulator

See the code: https://github.com/rmboyce/orbit-sim

Simple orbit simulator that ignores interactions between planets. It finds where the planets should be and what speeds they should be moving at by solving the two-body problem with the Kepler Equation. Uses Newtonâ€™s Method to calculate a numerical approximation.

Control by using four clicks: The first two clicks set the orbit, and the last two set the planet radius.
